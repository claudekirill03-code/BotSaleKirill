# Korаgаж Sales Coach Bot 🛋️

Telegram бот для анализа диалогов менеджеров Корагажа и помощи в закрытии клиентов на встречу в офис.

## Что умеет делать бот?

✅ Анализирует скриншоты переписки менеджера с клиентом  
✅ Выявляет ошибки в диалоге  
✅ Дает конкретные рекомендации что делать дальше  
✅ Предлагает готовые варианты сообщений  
✅ Озвучивает ответы голосом  
✅ Распознает голосовые сообщения и помогает с текстом  
✅ **НОВОЕ**: Логирует все анализы в базу данных  
✅ **НОВОЕ**: РОП может смотреть статистику использования (`/stats`)  

## Требования

- Python 3.11+
- Telegram Bot Token (от @BotFather)
- Anthropic Claude API Key
- OpenAI API Key (для голоса)
- Railway аккаунт (для хостинга)

## Установка локально

```bash
# Клонируй репо
git clone https://github.com/YOUR_USERNAME/koragaj-bot.git
cd koragaj-bot

# Создай виртуальное окружение
python -m venv venv
source venv/bin/activate  # На Windows: venv\Scripts\activate

# Установи зависимости
pip install -r requirements.txt

# Скопируй .env.example в .env и заполни переменные
cp .env.example .env

# Запусти бота
python bot.py
```

## Переменные окружения

Создай файл `.env` с такими переменными:

```
BOT_TOKEN=твой_токен_от_botfather
ANTHROPIC_API_KEY=твой_ключ_anthropic
OPENAI_API_KEY=твой_ключ_openai
ALLOWED_USER_ID=0
ADMIN_ID=твой_telegram_id
```

- **ALLOWED_USER_ID**: ID менеджеров которые могут использовать бота (0 = все)
- **ADMIN_ID**: твой Telegram ID для команды `/stats` (только ты сможешь смотреть статистику)

### Получение ключей:

1. **BOT_TOKEN**: напиши @BotFather в Telegram, выполни /newbot
2. **ANTHROPIC_API_KEY**: https://console.anthropic.com → API Keys
3. **OPENAI_API_KEY**: https://platform.openai.com → API keys

## Запуск на Railway

1. Создай GitHub репо и залей туда этот код
2. Зайди на https://railway.app
3. Нажми "New Project" → "Deploy from GitHub"
4. Выбери репо
5. Railway автоматически найдет Dockerfile и запустит
6. В настройках проекта добавь переменные окружения:
   - BOT_TOKEN
   - ANTHROPIC_API_KEY
   - OPENAI_API_KEY
   - ALLOWED_USER_ID
   - ADMIN_ID (твой Telegram ID для /stats)

## Использование

1. Найди бота в Telegram (по юзернейму)
2. Нажми /start
3. Скидывай скрины переписки или голосовые сообщения
4. Бот проанализирует и даст рекомендации

### Команда /stats (для РОП)

Нажми `/stats` чтобы увидеть статистику использования бота:
- Сколько анализов сделано сегодня (по типам)
- Статистика за неделю
- Общие цифры (кол-во менеджеров, всего анализов)

**Доступ**: Только для ADMIN_ID (твой Telegram ID)

## Типовые ошибки менеджеров (которые бот ловит):

❌ Не показывает уникальность Корагажа (дизайн разрабатываем сами)  
❌ Просто называет цену без подведения к ценности  
❌ Не предлагает рассрочку при возражении "дорого"  
❌ Не пытается закрыть на встречу в офис  
❌ Молчит на игнорящих лидов (нужна активизация)  
❌ Робкий, без уверенности в уникальности
